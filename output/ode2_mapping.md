# Fellow Ode Gen 2 — Grind Size Mapping Reference

> **Disclaimer:** All grind mappings in this document are community estimates based on forum discussions, third-party measurements, and user reports. None come from the V60 experts profiled in this guide. Fellow does not publish official micron specifications. Always dial in by taste.

## Grinder Specifications

| Spec | Value |
|---|---|
| Burr type | 64mm stainless steel flat burrs |
| Burr designer | Nick Terzulli (Fellow VP of R&D), with input from Lance Hedrick |
| Total grind settings | 31 stepped (NOT half steps — ~3 clicks per numbered position) |
| Dial range | 1 (finest) – 11 (coarsest) |
| Sub-step notation | Community uses X.0, X.1, X.2 (e.g., 4.0, 4.1, 4.2) |
| Approximate micron range | ~250–1,160 (community-measured) |
| Grind speed | <20 seconds for 30g at medium |
| Hopper | Single-dose, 100g catch cup |

## Brew Method Mapping

### Fellow Official (Hopper Lid Guide)

| Brew Method | Dial Setting |
|---|---|
| Prismo / AeroPress (espresso-style) | 1–3 |
| AeroPress (standard) | 3–6 |
| Pour-over (V60, Chemex) | **4–8** |
| Cupping | 6–7 |
| Batch brew / electric drip | 7–9 |
| French Press | **8–10** |
| Cold Brew | 9–11 |

**Fellow's recommended starting point for V60: Setting 5.**

### Community Consensus (Reddit, Home-Barista, Coffeetime forums)

| Brew Method | Community Range | Notes |
|---|---|---|
| V60 (fine technique) | 3.5–4.5 | Small doses, Hoffmann-style, high agitation |
| V60 (standard) | **4.5–6.0** | Most common range; start at 5 |
| V60 (coarse pulse) | 5.5–6.5 | Kasuya 4:6, Winton, Rolf methods |
| Chemex | 5–7 | Pour-over, not immersion |
| French Press | 8–10 | True immersion |
| Cold Brew | 9–11 | Coarsest settings |
